# To-Do List Web App

This is a simple To-Do List web application built with PHP and MySQL. Users can add, edit, delete, and mark tasks as done.

## Features

- User can add tasks.
- User can mark tasks as "Done."
- User can edit and delete tasks.
- Simple and responsive design using Bootstrap.

## Requirements

- PHP 7 or higher
- MySQL database
- Web server (e.g., Apache)

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/todolist-web-app.git
   ```

2. Set up the database:
   - Create a database `db_task` in MySQL.
   - Set up the `task` table.

3. Configure database connection:
   - Edit the `conn.php` file to match your database credentials.

4. Run the app:
   - Upload the project files to your server or run it locally.
   - Access the app in your browser.

## How to Use

- Register or log in to access the task list.
- Add tasks to your to-do list.
- Mark tasks as done or delete them.
